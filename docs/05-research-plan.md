# 05 Research Plan: Paper Reproducibility and Domain-Relativity Manuscript

## Project status

PoemForge Phase A has changed category.

The original project tested whether compression-progress could act as a label-free aesthetic value functional for poetry. The result does not support that claim. Generic external literary domains fail to recover contemporary human poetic appraisal and can anti-align with human ratings. Human-labeled contrastive domains do recover appraisal structure. Once such domains are constructed, compression, TF-IDF, and sentence-embedding readouts recover similar signal under full controls.

The project is now two things:

1. A paper about domain-relative supervised value signals.
2. A possible future product direction centered on domain construction, not label-free value discovery.

This document covers the paper path only.

---

# 1. Paper thesis

## One-sentence thesis

Generic compression-progress over external literary domains does not recover human poetic appraisal; human-labeled contrastive domains do, and once such domains are constructed, multiple readout metrics recover similar signal, showing that the value signal lives primarily in the domain rather than in any uniquely privileged metric.

## Terminal claim

This experiment does not show that compression-progress mechanizes aesthetic value label-free. It shows that aesthetic value enters through the construction of D. Generic D fails; human-labeled D works. Once D is human-shaped, compression, TF-IDF, and embeddings recover similar signal under full controls. Compression is competitive and sometimes higher by mean, especially through matched-other subtraction, but this sample cannot resolve a unique compression advantage. The robust contribution is domain-induced supervised value, not compression-specific value discovery.

# 2. Current evidence status

## Strong claims

### C1. Generic D fails

Generic external literary domains do not recover contemporary human poetic appraisal and can anti-align with human ratings.

Status: strong within current dataset and current analyses.

### C2. Preference-shaped D works

Human-labeled high/low domains recover appraisal structure in held-out settings.

Status: strong within current dataset.

### C3. Metric convergence supports domain-relativity

Compression, TF-IDF, and sentence embeddings converge under full controls.

Status: strong conceptual result within current dataset, but needs external validation before strong submission framing.

### C4. Label-free compression value is not supported

The original label-free compression-progress aesthetic value claim is not supported.

Status: strong.

---

## Moderate claims

### C5. Compression is competitive

Compression is often higher by mean in structural settings, especially before full stacking.

Status: descriptive, not established as statistically superior under item-level uncertainty.

### C6. Matched-other subtraction may reveal a compression-specific interaction

Matched-other subtraction helps compression at the run level and remains higher by mean than explicit normalization variants, but poem-level bootstrap cannot resolve the mechanism.

Status: open thread.

---

## Not established

### N1. Compression uniquely beats TF-IDF or embeddings

Not established. Poem-level bootstrap confidence intervals include zero.

Correct phrasing:

> Compression’s point estimates lean positive in some partially controlled structural settings, but n = 36 cannot resolve whether the advantage is stable.

### N2. Matched-other is definitely more than normalization

Not established. The matched-other interaction is unresolved.

Correct phrasing:

> Matched-other subtraction is the one plausible compression-specific mechanism, but this dataset cannot confirm or reject it.

### N3. Human Surprise equals LM unpredictability

Not supported. Higher-rated poems appear more LM-predictable, not less.

Correct phrasing:

> Human Surprise in this dataset is not equivalent to language-model unpredictability. It may reflect successful novelty within fluent form, but that mechanism remains interpretive.

# 11. External validation gate

The current result is based primarily on n = 36 poems from one rated corpus. Before aiming for a strong submission, the core triad should be tested on at least one additional rated creative-writing dataset.

Core triad:

```text
1. generic D fails or misaligns
2. preference-shaped D works
3. multiple readouts converge under controls
```

If the triad replicates:

```text
the paper becomes substantially stronger
```

If it does not replicate:

```text
the paper becomes a smaller, honest Chaudhuri-domain pilot
```

External validation should use the same pipeline without special casing.

Validation protocol:

```text
1. Add new dataset loader.
2. Normalize to item_id, text, target columns.
3. Run identical domain construction.
4. Run identical readouts.
5. Run identical controls.
6. Run identical bootstrap.
7. Compare triad status.
```

No new metric should be introduced until after the validation attempt.
