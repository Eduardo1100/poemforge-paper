# CLAUDE.md: Paper Reproducibility Instructions

This repo is a paper artifact, not a product repo.

## Mission

Regenerate every reported number from raw or frozen input data using a clean, ordered, idempotent pipeline.

Do not optimize for making compression look good. Optimize for making every claim auditable.

## Core result to preserve

```text
generic D fails
human-shaped D works
metrics converge
compression is competitive but not uniquely established
value lives in the domain
```

## Required invariants

1. Every manuscript claim must map to a generated output in `docs/claims_manifest.csv`.
2. Every output file used in the manuscript must be hashable and reproducible.
3. No paper table or figure should be hand-edited.
4. Do not quote old synopsis values unless regenerated by the final pipeline.
5. Do not add new controls to rescue compression-specificity on the n=36 dataset.
6. Do not promote run-level p-values over poem-level uncertainty.
7. External validation should use the same pipeline without special casing.

## Pipeline order

```text
00_prepare_data
10_build_domains
20_score_readouts
30_compute_controls
40_residualize
50_permutation
60_bootstrap
90_make_tables
91_make_figures
99_diff_against_scaffold
```

## First milestone

Implement `pipeline/run_all.py` so `make reproduce` runs all stages in order. It is acceptable for early stages to fail loudly with "not implemented" if the stage has not yet been ported, but the pipeline order must be explicit.

## Readout abstraction

Compression, TF-IDF, and embedding readouts must share one interface:

```python
class Readout:
    def fit_domain(self, positive_examples, negative_examples, metadata=None): ...
    def score_candidate(self, candidate_text: str) -> dict: ...
    def score_batch(self, candidates) -> list[dict]: ...
```

This code structure reflects the paper's methodological claim: different readouts are lenses over D.

## Paper framing

Lead with the positive result:

- Generic world/literary D can misalign with human preference.
- Human-labeled D recovers appraisal.
- Multiple unrelated readouts converge once D is human-shaped.

Do not frame the paper primarily as "compression failed." The scientific result is domain-relativity.
